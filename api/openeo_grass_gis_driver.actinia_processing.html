openeo_grass_gis_driver.actinia_processing.zonal_statistics.create_process_chain_entry(input_name, polygons)[source]¶

Create a Actinia command of the process chain that computes the regional statistics based on a strds and a polygon.

The computational region will be set to the vector map, the previous region will be saved and after processing restored. A mask will be set that uses the vector file as input. This mask will be removed in the end.

Parameters:
  • input_name – The name of the strds
  • polygons – The URL to the vector file that defines the regions of interest
Returns:

A Actinia process chain description
